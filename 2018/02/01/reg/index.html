常用正则表达式整理

Feb 1, 2018

将常用的正则表达式整理如下。

常用正则表达式整理

正则元字符

字符 描述
* 匹配前面的字表达式零次或者多次等价于{0,}
+ 匹配前面的字表达式一次或者多次等价于{1,}
? 匹配前面的字表达式零次或者一次等价于{0,1}

整数(不限制正负)

/^-?\d+$/; // 不限制长度

整数(> 0)

/^[1-9]\d*$/; // 不限制长度
/^[1-9]\d{0,5}$/;// 限制长度

整数(> 0 && <= 100)

/^(100|([1-9]{1}\d?))$/

浮点型(> 0 保留2位小数)

/^(([1-9]{1})(\.\d{1,2})?|(0(\.\d{0,1}[1-9])))$/


/^(([1-9]{1}[0-9]{0,5})(\.\d{1,2})?|(0(\.\d{0,1}[1-9])))$/

浮点型(>= 0 保留2位小数)

/^(([1-9]{1})(\.\d{1,2})?|(0(\.\d{0,1}[1-9])?))$/

字符串开头不能空格 中间可以是空格/数字/字母/中文/非特殊字符结尾

/^[\w\u4e00-\u9fa5]{1}[\s\w\u4e00-\u9fa5]{0,8}$/
